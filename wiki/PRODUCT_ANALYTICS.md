# Product Analytics & User Insights

Understanding user behavior and product impact.

---

## 📊 Analytics Types

**Behavioral**: What do users do?
**Engagement**: How involved are they?
**Retention**: Do they come back?
**Monetization**: Do they pay?
**Funnel**: Drop-off at stages?

---

## 🎯 Key Metrics

**DAU/MAU**: Daily/Monthly Active Users
**Retention Rate**: % users returning
**Churn Rate**: % users leaving
**LTV**: Lifetime value per user
**CAC**: Cost to acquire user
**Engagement**: Time, frequency, depth

---

## 🔄 Analytics Pipeline

```
Event collection
    ↓
Aggregation
    ↓
Analysis
    ↓
Visualization
    ↓
Insights
    ↓
Action
```

---

## 🧪 Experimentation

**A/B Test**:
- Control group: Current feature
- Variant group: New feature
- Measure metric impact
- Declare winner

**Success Criteria**:
- Sample size: Enough statistical power
- Duration: Enough time for effects
- Significance: 95% confidence
- Practical significance: Worth shipping

---

## 💡 Insight Examples

"Users with mobile app install 2x more likely to retain"
→ Action: Improve mobile onboarding

"Checkout abandonment 40% at shipping step"
→ Action: Simplify shipping
