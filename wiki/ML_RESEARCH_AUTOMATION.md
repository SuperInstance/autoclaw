# Machine Learning Research Automation

Using AutoClaw to accelerate ML research workflows.

---

## 🤖 ML Research Pipeline

```
1. Literature review (Researcher agent)
   ↓
2. Experimental design (Critic agent)
   ↓
3. Paper analysis (Distiller agent)
   ↓
4. Synthesis (Teacher agent)
   ↓
5. Knowledge base (Knowledge store)
```

---

## 📚 Research Tasks

- **Paper summarization**: Extract key contributions
- **Methodology comparison**: Find best approaches
- **Dataset discovery**: Identify public datasets
- **Benchmark tracking**: Monitor SOTA results
- **Citation analysis**: Find influential work
- **Trend detection**: Spot emerging directions

---

## 🔬 Experiment Analysis

- Parse experiment results
- Compare architectures
- Identify hyperparameter patterns
- Find performance trade-offs
- Spot reproducibility issues
- Document learnings

---

## 📊 Knowledge Extraction from Papers

- **Abstract → Key findings**
- **Methods → Reproducible steps**
- **Results → Benchmarks**
- **Related work → Context**
- **Limitations → Future directions**
