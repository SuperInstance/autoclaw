# Temporal Reasoning & Time-Series Analysis

Reasoning about time, causality, and sequences.

---

## ⏰ Temporal Concepts

**Point Events**: Happen at specific time
**Interval Events**: Occur over time period
**Sequences**: Order matters
**Causality**: X causes Y

---

## 🔄 Temporal Relationships

```
Before: Event A happens before Event B
After: Event A happens after Event B
During: Event A overlaps with Event B
Causes: Event A leads to Event B
Enables: Event A makes Event B possible
```

---

## 📊 Time Series Analysis

**Trend Detection**:
- Is value increasing/decreasing?
- How fast?
- Will it continue?

**Seasonality**:
- Recurring patterns?
- Daily/weekly/yearly cycles
- Anomalies relative to season

**Forecasting**:
- ARIMA models
- Exponential smoothing
- ML-based predictions

---

## 🔗 Causal Reasoning

**Correlation vs Causation**:
- Correlation: X and Y move together
- Causation: X causes Y
- Confounding: Z causes both

**Causal Inference**:
- Intervention analysis
- Counterfactual reasoning
- Effect sizes

---

## 📈 Applications

- Stock price prediction
- Disease progression
- Weather forecasting
- Event prediction
- Trend analysis
