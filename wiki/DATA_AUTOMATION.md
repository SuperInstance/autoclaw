# Data Automation & Spreadsheet Workflows

Automating data processing pipelines for spreadsheets, databases, and structured data.

---

## 🔄 CSV Processing Workflow

**Common Pattern**:
1. Read CSV file
2. Parse rows with validation
3. Transform/normalize data
4. Validate against schema
5. Load into system
6. Generate report

---

## 📊 Excel Integration

- Read workbooks
- Extract specific sheets
- Validate data types
- Handle merged cells
- Preserve formatting
- Output results

---

## ✅ Data Validation

- Type checking
- Range validation
- Format validation
- Cross-field validation
- Business rule validation
- Error collection and reporting
