# Ensemble Methods & Multi-Model Systems

Combining multiple agents and models for better results.

---

## 🎯 Ensemble Strategies

**Voting**: Majority decides
**Averaging**: Combine predictions
**Weighted**: Some models count more
**Stacking**: Models feed other models
**Bagging**: Different data subsets
**Boosting**: Focus on hard cases

---

## 🤖 Agent Ensembles

```
Task arrives
    ↓
Multiple agents: Researcher, Critic, Distiller, Teacher
    ↓
Each produces result
    ↓
Combine/vote
    ↓
Final answer
```

---

## ⚖️ Aggregation Methods

- **Simple averaging**: All equal weight
- **Weighted averaging**: Based on expertise
- **Voting**: Majority wins
- **Stacking**: Meta-learner combines
- **Rank aggregation**: Order-based combining
- **Bayesian**: Probabilistic combination

---

## 📊 When to Use Ensembles

- High-stakes decisions (combine voices)
- Diverse opinions (different perspectives)
- Model uncertainty (average multiple views)
- Edge cases (need multiple approaches)
- Quality improvement (reduce error)
