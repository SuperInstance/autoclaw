# Transfer Learning & Domain Adaptation

Applying knowledge from one domain to another.

---

## 🔄 Transfer Learning Concept

```
Pre-trained model (learned on task A)
            ↓
Fine-tune on task B
            ↓
Better performance with less data
```

---

## 🎯 Transfer Scenarios

**Same Domain, Different Task**:
- Medical diagnosis → Medical transcription

**Different Domain, Same Task**:
- ImageNet features → Custom images

**Cross-Domain**:
- English → German translation
- Product reviews → Review classification

---

## 🛠️ Techniques

**Feature Transfer**: Use learned features
**Task Transfer**: Adapt from related task
**Domain Adaptation**: Bridge domain gap
**Meta-Learning**: Learn to learn quickly

---

## 📊 Data Requirements

```
No transfer: Need 10k examples
With transfer: Need 1k examples (10x reduction)
```

---

## 💡 Applications

- Few-shot learning
- Domain adaptation
- Multi-task learning
- Personalization
