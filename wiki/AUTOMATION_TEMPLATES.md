# Reusable Automation Templates

Copy-paste workflows for common automation tasks.

---

## 📋 Template Catalog

### Research Template
```
crew add "Research [topic] focusing on [aspect]"
crew knowledge query "[topic]"
```

### Code Review Template
```
crew add "Review code in [file] for quality"
crew show <task_id>
```

### Data Processing Template
```
crew add "Process [data source] and extract [fields]"
crew knowledge query "[data type]"
```

### Tutorial Generation Template
```
crew add "Create tutorial on [topic]"
crew knowledge query "tutorial [topic]"
```

---

## ⚙️ Customization Guide

1. Replace placeholders
2. Adjust priority if needed
3. Add tags for filtering
4. Set confidence threshold
5. Execute and monitor
