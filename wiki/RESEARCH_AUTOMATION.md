# Research Automation Workflows

Automating literature review, synthesis, and knowledge building.

---

## 📚 Research Pipeline

```
1. Topic definition
2. Source identification
3. Paper collection
4. Content extraction
5. Synthesis
6. Knowledge base creation
7. Quality validation
```

---

## 🔍 Source Discovery

- Academic databases
- GitHub repositories
- Technical blogs
- Documentation
- News sources
- Company publications

---

## 📊 Content Extraction

**From papers**:
- Abstract/summary
- Key findings
- Methodology
- Citations
- Limitations

**From code**:
- Architecture
- Features
- Performance
- Limitations

---

## 🎯 Synthesis Techniques

- Compare findings across sources
- Identify agreements/disagreements
- Spot trends
- Highlight gaps
- Suggest future work
