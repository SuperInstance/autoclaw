# Prompt Engineering for Agents

Crafting effective prompts that guide agent behavior.

---

## 🎯 Prompt Structure

```
1. Role/Context: "You are a..."
2. Task: "Your job is to..."
3. Requirements: "Must include..."
4. Format: "Output as..."
5. Quality: "Be thorough..."
6. Constraints: "Don't..."
```

---

## 💡 Techniques

**Few-shot examples**: Show desired output
**Instruction clarity**: Be specific
**Constraint specification**: Set boundaries
**Output formatting**: Specify format
**Quality indicators**: Set standards

---

## 🔄 Iterative Refinement

1. Write initial prompt
2. Test on sample task
3. Analyze quality
4. Identify gaps
5. Refine prompt
6. Repeat until satisfied

---

## 📊 Prompt Metrics

- Output quality (1-5)
- Task completion rate
- Latency impact
- Cost per token
- Edge case handling
