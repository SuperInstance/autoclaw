# Recommendation & Personalization

Building recommendation systems with AutoClaw.

---

## 🎯 Recommendation Types

**Collaborative**: Based on similar users
**Content-based**: Similar to liked items
**Hybrid**: Combining approaches
**Knowledge-based**: Based on explicit rules
**Contextual**: Based on situation
**Temporal**: Trending now

---

## 🔄 Recommendation Pipeline

```
User data → Analysis → Generate → Rank → Personalize → Deliver
    ↓         ↓         ↓        ↓       ↓            ↓
History   Preferences Candidates Score  Format       Present
Behavior  Patterns    Similar    Rank   Customize    Track
Feedback  Trends      Popular    Filter Explain      Improve
```

---

## 📊 Recommendation Dimensions

- **Relevance**: How well matched?
- **Diversity**: Avoid repetition
- **Novelty**: Discover new items
- **Coverage**: Recommend everything
- **Serendipity**: Surprising but good
- **Explainability**: Why this?

---

## 📈 Success Metrics

- Click-through rate
- Conversion rate
- Engagement time
- User satisfaction
- Coverage
- Diversity
